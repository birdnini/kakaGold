# kakaGold
A lightweight global macro four-dimensional radar &amp; LSTM fire control system for gold price
📖 第一部分：Application Note (用户操作手册)
欢迎来到 Kaka-Terminal！这是一款运行在本地的轻量级、全自动全球宏观资金流向监控基站。它不仅能为你 24 小时盯盘，还能在你需要时，调用本地的 AI 算力为你测算未来的胜率。

1. 📦 部署与启动 (Installation)
本程序采用“绿色免安装（Portable）”架构设计：

将收到的压缩包解压到你的电脑任意全英文路径下（避免中文路径引发底层寻址错误）。

确保文件夹内包含 meme.jpg（托盘图标）和 coin.wav（警报音效）。

双击运行主程序 .exe，系统将自动在当前目录下生成 kaka_macro_log.csv（数据沉淀池）和配置文件。

程序支持**“老板键（Alt+X）”**或点击右上角关闭按钮，界面会瞬间隐形并潜伏在右下角系统托盘继续运行。

2. 📡 传感器矩阵模块 (Sensor Matrix)
系统接驳了全球四大核心资产的 API 总线，每 10 秒钟自动同步一次数据：

✿ 黄金 (国内预估)：避险情绪与抗通胀终极标的。

✿ WTI 原油 (主力)：全球通胀的底层引擎与工业血液。

✿ 美元指数 (DXY)：全球流动性基准电压。

✿ NVDA (算力信仰)：代表当代科技革命与风险偏好的最强风向标。

如何使用报警器？
在每个资产下方的 [跌破] 和 [涨破] 输入框内填入你关注的极值点（必须为纯数字）。当价格触底或突破时，系统会触发物理蜂鸣报警，并在右下角弹出警报弹窗。

3. 📊 示波器模块 (Waveform Oscilloscope)
点击 【📊 波形】 按钮，系统将调取本地积累的 CSV 数据，渲染出高精度的四维动态时序波形。

极客建议：由于采用差分波动记录法，请先点击【▶ 锁定采集】让系统在后台运行半天以上，积累至少 5 条以上数据后再生成波形，否则图表会因为数据量过少而拒绝渲染。

4. 🧠 终极火控系统 (AI Fire Control System)
这是本基站的核心大脑。点击底部粉色的 【⚡ AI 测算】 按钮，系统将唤醒内置的 64 核心 LSTM 神经网络。

动能雷达：AI 在吞下数据前，会自动计算 MACD (多普勒加速度)、RSI (弹簧过载传感器)、对数收益率 和 金油比 等 9 维高阶特征。

安全仓位建议：AI 算出的胜率将直接输入“凯利公式（Kelly Criterion）”。为了保护你的本金，系统严格执行华尔街机构级的**“1/4 凯利限流 + 5% 绝对物理熔断”**双重保险丝法则。

指令判读：

[ BUY ] / [ SELL ]：明确的趋势信号，并附带安全仓位比例（最高不超过 5%）。

[ IDLE ]：当前市场处于底噪震荡区，信号极弱，系统强行锁死扳机，建议空仓观望。
